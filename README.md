# 🛰️ SatConnect Intelligence Platform
### Satellite Internet Coverage Gap Analysis — Starlink vs OneWeb vs ISRO

## 📊 Project Overview
A comprehensive 5-page Power BI dashboard analyzing global 
satellite internet coverage gaps, provider comparisons, 
and digital divide insights.

**Microsoft Elevate — AICTE Internship Project 2026**
**Student:** Pracheta Patel (211200111677)

## 🎯 Key Findings
- Global Coverage 2023: 38.68% (up from 8.88% in 2000)
- Starlink dominates with 88% market share (5,500 satellites)
- India average: 52.4% vs 75% national target by 2030
- Digital divide: 70+ point gap between high/low income countries
